# **App Name**: Dream Weaver AI

## Core Features:

- Dream Input: Users can input details about the person they saw in their dream, focusing on details other than the face (clothing, environment, etc.).
- AI Reference Generation Tool: Generates a reference about who this person could be, using AI to match the description with known profiles. The LLM uses a tool to determine if enough information has been provided by the user to incorporate additional possible traits for the dream character
- Reference Display: Presents possible profiles of the person from the dream, ordered by relevance.
- Additional Details Input: Allows users to add more details if the initial references are not satisfactory.

## Style Guidelines:

- Primary color: Soft Lavender (#D0B8FF) to evoke a dreamlike, mysterious feeling.
- Background color: Very Light Grey (#F0F0F5), nearly white, to provide a clean, calming backdrop.
- Accent color: Muted Teal (#73A7C3) for interactive elements, providing a subtle contrast.
- Font: 'Alegreya', a serif font with an elegant, intellectual, contemporary feel; suitable for headlines or body text.
- Use minimalist, symbolic icons to represent different aspects of dreams and descriptions.
- Clean and simple layout, focusing on ease of use and clarity. Prominent search bar and clear display of potential matches.
- Subtle transitions and animations to create a dreamy, ethereal feel.
